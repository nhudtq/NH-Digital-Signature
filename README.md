# Các bước cài đặt
## setting python
`
    brew install python
    apt install python3
    yum install python3

`
## Install Python-pip

## setting Django
`
    sudo pip install django
`

## Tạo project python_rsa
`
    django-admin startproject python_rsa
`

## Run 
`
    - pip install pycryptodome
    - python manage.py runserver
    - python manage.py runserver 8080
`
